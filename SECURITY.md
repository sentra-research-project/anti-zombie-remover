# Contributing to Anti-Zombie Remover

First off, thank you for considering contributing to this project! Since this is a community-focused mod, our philosophy is simple: **Openness and Collaboration.**

## How Can You Contribute?

* **Fork & Modify:** You are free (and encouraged!) to fork this repository, modify the code, and release your own versions.
* **Pull Requests:** If you fix a bug or add a cool feature that benefits everyone, feel free to send a Pull Request.
* **Issues:** Found a bug? Open an issue to let us know.

## Re-distribution & Credit

You can use this code in your own projects or modpacks. We only ask for one thing (as per the MIT License):

* **Give Credit:** Please keep the original license file or mention "Original work by SENTRA Research Project / ed.husserl" in your project.
